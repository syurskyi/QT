# PyQt5-Game

Simple game (Bomberman clone) created with PyQt5.  
It is still in development mode.

How to play:  
Move - arrow keys  
Space - drop bomb  
You can control another bot, if you click on him.  
Bots move completely random

Upcoming features:  
-TCP/IP connect
